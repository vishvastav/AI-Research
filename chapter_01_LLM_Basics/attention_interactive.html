Scaled Dot-Product Attention  ·  Attention(Q, K, V) = softmax(QKT / √dk) · V
Your sentence
1

Tokenize

sentence → tokens

Split the sentence into pieces (here: lowercase words). Each token gets an integer ID from a vocabulary. Real models use BPE / WordPiece — sub-word pieces — so unseen words still work.

2

Embed

E ∈ ℝn × dmodel

Each token ID maps to a vector of d_model numbers. Vectors are learned in real training — here we derive each one deterministically from the token text so the same word always lands at the same point in space.

Rows = tokens. Columns = the abstract "features" the model has learned. Similar words have similar rows.
3

Project to Query, Key, Value

Q = E·WQ  K = E·WK  V = E·WV

Three learned weight matrices W^Q, W^K, W^V (each d_model × d_k) turn each embedding into three new vectors. Query = "what I want", Key = "what I offer", Value = "what I carry".

K matrix

V matrix

4

Raw attention scores: Q · KT

scores ∈ ℝn × n

Dot-product every Query with every Key. scores[i][j] = how much token i wants from token j. Big number = strong match.

5

Scale by √dk

scores / √dk

Without scaling, large dk creates huge dot-products → softmax saturates → gradients vanish. Dividing by √dk keeps values in a useful range.

6

Softmax → attention weights

A = softmax(scores)

Apply softmax row by row. Each row now sums to 1 — interpret as percentages: how much of token i's attention goes to each other token.

low medium high
Read row-wise: row "cat" tells you what "cat" pays attention to. Each row sums to 1.00.
7

Multiply by V → output

output = A · V

Each token's new vector = weighted sum of all Values, using attention weights. The word is now contextualized — it carries information from the words it attended to.

8

Multi-Head Attention

Concat(head1, ..., headh) · WO

Run h separate attention computations in parallel — each with its own W^Q, W^K, W^V. Each head learns a different relationship pattern (one might track subject↔verb, another track noun↔modifier, etc).

Final output (all heads concatenated)

Real transformer: this stack of self-attention + feed-forward is repeated 12 to 96 times. The same mechanism, learned to do different jobs at different depths.
